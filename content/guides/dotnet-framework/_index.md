The .NET Framework getting started guide teaches you how to create a containerized .NET Framework application using Docker and Windows containers. In this guide, you'll learn how to:

- Containerize and run a .NET Framework application
- Set up a local environment to develop a .NET Framework application using containers
- Run tests for a .NET Framework application using containers
- Configure a CI/CD pipeline for a containerized .NET Framework application using GitHub Actions
- Deploy your containerized application locally to Kubernetes to test and debug your deployment

After completing the .NET Framework getting started modules, you should be able to containerize your own .NET Framework application based on the examples and instructions provided in this guide.

Start by containerizing an existing .NET Framework application.
